# Scripts Context

This folder contains utility scripts and automations that are useful across the entire Open Claw repository.

## Guidelines

1. **Global Utilities:** Scripts here should serve the project as a whole (e.g., build scripts, repository maintenance, mass-execution tasks).
2. **Local Scripts:** If a script is only relevant to a single Skill or Plugin, it should be placed inside that Skill's or Plugin's own folder, not here.
