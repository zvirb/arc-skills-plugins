# Antigravity Workspace Agents

This file lists the active agents in this workspace and their primary responsibilities. Use this to determine which agent to delegate tasks to.

## 1. Main Agent (`main`)
* **Role:** Primary entry point and system coordinator.
* **Responsibilities:** Initial task triage, overall system health monitoring, and high-level project management.

## 2. Researcher (`researcher`)
* **Role:** Deep information gathering and analysis.
* **Responsibilities:** Web searching, technical documentation review, and investigating complex APIs or dependencies.

## 3. Coder (`coder`)
* **Role:** Engineering and implementation lead.
* **Responsibilities:** Writing clean, testable TypeScript/JavaScript code, developing plugins, and performing surgical bug fixes.

## 4. Summarizer (`summarizer`)
* **Role:** Information condensation and insight extraction.
* **Responsibilities:** Processing large logs, summarizing long discussions, and creating concise executive reports.

## 5. Operator (`operator`)
* **Role:** System stability and maintenance specialist.
* **Responsibilities:** Running self-healing workflows, managing heartbeats, and ensuring the OpenClaw gateway remains stable.
